注意事项:
* 完完全全的原版插入转译器不干扰原计算模型实现中文支持,原版的BUG请到原作者的GITHUB提交

* 国际服版天赋树,装备物品搜索请用英文,不支持中文
* 双版数据不互通,请勿串用

FAQ:

>> POB 提示更新,要不要点?
* POB内的提示更新属于 POB 自己的更新,点击更新=更新 POB

>> 原来的 POB BD 存档怎么放到 PoeCharm
1)打开原来 POB 目录下的 builds 目录.
2)PoeCharm Pob 启动图标 右键"打开存档"
3) 步骤 1. 复制到步骤 2. <提示: 国服 国际服版本存档不互通,请勿串用>

>> 能不能更换字体
* 可以更换 字体文件放在软件目录 PoeCharm/Fonts 目录下,替换 FZ_ZY.ttf

>> 国服(国际服)链接导入国际服(国服)只有天赋没有装备?
* 国服 国际服版本存档不互通,请勿串用

>> 怎么更新软件啊 老是更新失败！！！
* 墙或者GitHub有改动,POB(国际服版)无法从外网(github)获取更新文件(搭(fang)梯(qiang)可解))

>> 怎么添加别的版本的POB到PoeCharm?
* 复制 POB 整个目录到 PoeCharm 主程序目录下,重新启动 PoeCharm 即可
